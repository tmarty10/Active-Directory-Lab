## Setup Overview

This folder documents the full environment setup for the project, including both the help desk system and the underlying domain infrastructure.

The setup is divided into two main components:

- **osTicket Deployment:**  
  Installation and configuration of a web-based help desk system using IIS, PHP, and MySQL on a Windows 11 virtual machine. This includes configuring the web server, database, and application to create a fully functional ticketing platform.

- **Active Directory Environment:**  
  Configuration of a Windows Server 2022 domain controller and the integration of endpoint devices into the domain. This includes domain creation, organizational unit structure, and endpoint configuration for centralized management.

Together, these components simulate a real-world enterprise environment where users can submit IT support tickets from domain-joined devices, and administrators can manage both infrastructure and support operations in a centralized system.
